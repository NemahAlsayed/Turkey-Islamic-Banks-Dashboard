# Türkiye Participation Banking Intelligence Dashboard  
**Interactive Islamic Finance Analytics (2014–2025)**  
Built by **Nemah Alsayed** — Multilingual MIS Specialist & Islamic Finance Analyst.

---

## 📌 Overview  
This dashboard provides a comprehensive, interactive analysis of Türkiye’s **Participation Banking (Katılım Bankacılığı)** sector using data from:

- TKBB (Türkiye Katılım Bankaları Birliği)  
- BDDK (Banking Regulation and Supervision Agency)  
- Participation bank annual reports  
- Macroeconomic indicators (CPI, M2, sectoral financing)  
- ESG & climate metrics  
- Islamic financing product portfolios  

The dashboard is hosted on **GitHub Pages** and built using **Plotly**, **HTML**, **CSS**, and **JavaScript**.
